The Problem & Solution

Why ShopSense?

ShopSense is a recommendation system designed to solve the "Cold Start" problem in e-commerce. While traditional Collaborative Filtering (KNN) offers high accuracy, it fails with new products.

Our Hybrid Architecture (CBF + KNN) combines the best of both worlds: ensuring 100% item coverage using metadata while leveraging user behavior for personalized discovery.

Solves Cold Start

Immediate visibility for new inventory.

High Accuracy

Maintains 84%+ prediction accuracy.

Methodology

1. Data Collection

Acquisition of the Amazon Products Dataset (390k+ rows) containing metadata and ratings.

2. Data Preprocessing

Cleaning, TF-IDF Vectorization, and generation of 12k Synthetic Users.

3. Model Development

Training 7 algorithms including CBF, KNN, SVD, MLP, XGBoost, and Hybrid models.

4. Prototype

Development of the "ShopSense" web application using Streamlit.
